Fairway : si le tee shot a atteint le fairway, ou pas (1 ou 0). Doit être 0 s'il s'agit d'un Par 3
Green : nombre de coups pour atteindre le green. (GIR : Green In Regulation)
Putts : nombre de puts pour terminer le trou. Attention : il faut être sur le green (n'est donc PAS le nombre de strokes avec le putter !)
Sand : nombre de coups pour sortir d'un bunker.
